# PixelCut Pro Studio

PixelCut Pro Studio is a browser-based image editing and background removal tool built with pure HTML, CSS, and JavaScript.

The project provides a modern, responsive interface that allows users to upload images, remove backgrounds manually, edit photos, and download transparent PNG files directly from their browser without installing any software.

---

## Features

### Background Removal

* Magic Wand Background Remover
* Manual Eraser Tool
* Adjustable Tolerance Control
* Adjustable Brush Size
* Transparent PNG Export

### Image Editing

* Crop Tool
* Rotate Image
* Flip Image
* Brightness Adjustment
* Contrast Adjustment
* Saturation Adjustment

### User Interface

* Modern Glassmorphism Design
* Responsive Layout
* Landing Page with Hero Section
* Smooth User
